# fukusuke-sushi

## Frontend
- React + Vite + TS
- Node version: v22.14.x


- Stack MERN

## Como ejecutar el Frontend
Entrar a la carpeta de /frontend

Si es la primera vez, hacer:

`npm install`

Si no, solo ejecutar:

`npm run dev`


# Deploy gh-pages
Para Actualizar la web en gh-pages ejecutar el siguiente comando:

`npm run deploy`
